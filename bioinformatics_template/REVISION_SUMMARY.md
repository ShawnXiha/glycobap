# 论文修改总结

## 修改日期: 2025-03-21

## 修改内容

### 1. 新增 "Evaluation Metrics" 子章节 (Methods)

**位置:** Methods 部分，Software Dependencies 之后

**修改内容:**
- 明确定义了三个评估指标的计算方式
- 关键澄清：**所有指标均基于单糖残基/单个糖苷键级别计算，而非整个分子结构**
- 提供了清晰的数学公式定义
- 给出了具体计算示例

**新增公式:**

1. **Epimer Accuracy (差向异构体准确率)**
   - 定义：正确识别的单糖残基数量 / 基准集中的单糖残基总数
   - 澄清：正确识别意味着CCD代码完全匹配（单糖类型、异头碳构型、绝对构型）

2. **Anomeric Accuracy (异头碳准确率)**
   - 定义：正确识别的糖苷键异头碳构型数量 / 基准集中的糖苷键总数
   - 澄清：aldoses使用C1作为异头碳，ketoses/sialic acids使用C2

3. **Linkage Accuracy (糖苷键准确率)**
   - 定义：正确识别的糖苷键连接位置数量 / 基准集中的糖苷键总数
   - 澄清：包括供体碳原子位置和受体氧原子位置

### 2. 修复 "Error Analysis" 数据逻辑 (Results)

**问题:**
- 原文声称 2/50 (4%) 结构需要人工干预
- 但 Table 2 报告 Overall stereochemistry >98%
- 逻辑矛盾：如果4%失败，准确率不应超过96%

**修复方案:**

1. **明确样本处置:**
   - 总样本：50个结构
   - 成功处理：48个 (96%)
   - 需人工干预：2个 (4%)，均为不支持的CCD条目

2. **澄清准确率计算基准:**
   - Table 2 中的准确率（98.5%, 98.2%, 96.8%）是基于 **成功处理的48个结构**
   - 新增整体准确率计算：
     - Epimer整体正确率: 94.0% (47/50)
     - Anomeric整体正确率: 94.0% (47/50)
     - Linkage整体正确率: 92.0% (46/50)

3. **新增 Table: Sample Disposition and Error Breakdown**
   - 清晰展示样本分类和错误分布
   - 区分"不支持CCD"与"算法错误"

4. **详细错误分析:**
   - Epimer错误: 1例 (Gal误识别为Glc)
   - Anomeric错误: 1例 (α误为β)
   - Linkage错误: 2例 (α2-8唾液酸连接)
